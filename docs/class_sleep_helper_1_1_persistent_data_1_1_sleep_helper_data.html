Detailed Description

Structure saved to the persistent data file (binary)

It must always begin with the SavedDataHeader (16 bytes)!

You can expand the structure later without incrementing the version number. Added fields will be initialized to 0. The size is limited to 65536 bytes (uint16_t maximum value).

Since the SleepHelperData structure is always stored in RAM, you should not make it excessively large.
